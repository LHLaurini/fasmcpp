Parameters
[in]memoryAmount of memory to allocate for the assembler
[in]predefinitionsMemoryAmount of memory to allocate for storing predefinitions
Note
predefinitionsMemory is only used for 64-bit platforms and is ignored otherwise.
Warning
Some platforms allow only one Assembler to be constructed and will throw an exception otherwise.
Exceptions
std::runtime_errorif failed to allocate memory or if the binary was not found.
std::length_errorif the requested amount of memory cannot fit in 32-bit addressable memory.
